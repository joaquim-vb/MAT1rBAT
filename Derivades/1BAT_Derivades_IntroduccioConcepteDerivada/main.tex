\section*{Conceptes previs: La Taxa de Variació Mitjana (TVM)}

Recordem que la \textbf{Taxa de Variació Mitjana} d'una funció $f(x)$ en un interval $[a, b]$ mesura la variació relativa de la funció respecte a la variable independent:
$$TVM[a, b] = \frac{f(b) - f(a)}{b - a}$$
Geomètricament, aquest valor coincideix amb el \textbf{pendent de la recta secant} que passa pels punts $(a, f(a))$ i $(b, f(b))$.

\section*{Càlcul algebraic: La definició de derivada}

Recordem que la derivada d'una funció $f(x)$ en un punt $x_0$ es defineix com el valor del següent límit:
\[ f'(x_0) = \lim_{h \to 0} \frac{f(x_0+h) - f(x_0)}{h} \]

\subsection*{Exercici 3. Càlcul de derivades mitjançant el límit}

    \item \textbf{Exemple resolt.} Donada $f(x) = 3x - 2$, calcula $f'(1)$:
    
    \textit{Resolució:}
    \begin{itemize}
        \item Identifiquem els elements: $x_0 = 1$, $f(x_0) = f(1) = 3(1)-2 = 1$.
        \item Calculem $f(x_0+h) = f(1+h) = 3(1+h) - 2 = 3 + 3h - 2 = 1 + 3h$.

        \[ f'(1) = \lim_{h \to 0} \frac{(1 + 3h) - (1)}{h} = \lim_{h \to 0} \frac{3h}{h} = \lim_{h \to 0} 3 = \mathbf{3} \]

\noindent\textbf{Resolució detallada de l'Exercici 4.4 (Més pràctica de funcions derivades)}

A continuació es detalla el càlcul de la derivada utilitzant la definició mitjançant el límit per a les dues funcions proposades.

\vspace{0.5cm}
\textbf{a) $f(x) = 2x^2 - 5$}

\begin{enumerate}
    \item \textbf{Plantegem el límit:}
    \[ f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h} \]
    
    \item \textbf{Substituïm la funció:}
    \[ f'(x) = \lim_{h \to 0} \frac{[2(x+h)^2 - 5] - [2x^2 - 5]}{h} \]
    
    \item \textbf{Desenvolupem el quadrat del binomi:}
    Recordem que $(a+b)^2 = a^2 + 2ab + b^2$. Per tant, $(x+h)^2 = x^2 + 2xh + h^2$.
    \[ f'(x) = \lim_{h \to 0} \frac{[2(x^2 + 2xh + h^2) - 5] - [2x^2 - 5]}{h} \]
    
    \item \textbf{Multipliquem i traiem parèntesis:}
    \[ f'(x) = \lim_{h \to 0} \frac{2x^2 + 4xh + 2h^2 - 5 - 2x^2 + 5}{h} \]
    
    \item \textbf{Simplifiquem termes al numerador:}
    Els termes $2x^2$ i $-2x^2$ s'anul·len, igual que $-5$ i $+5$.
    \[ f'(x) = \lim_{h \to 0} \frac{4xh + 2h^2}{h} \]
    
    \item \textbf{Traiem factor comú i resolem el límit:}
    \[ f'(x) = \lim_{h \to 0} \frac{h(4x + 2h)}{h} = \lim_{h \to 0} (4x + 2h) = 4x + 2(0) = \mathbf{4x} \]
\end{enumerate}


\vspace{0.5cm}
\textbf{b) $f(x) = (x + 2)^3$}

\textit{Nota prèvia: Com desenvolupar $(x+h+2)^3$?}
Per evitar confusions, ho farem pas a pas. Considerarem $(x+2+h)^3$ i ho separarem com a la multiplicació d'un binomi al quadrat pel mateix binomi: $[(x+2)+h]^2 \cdot [(x+2)+h]$. 
Per fer-ho més senzill, podem agrupar $x+2 = A$. Així hem de calcular $(A+h)^3$.
\begin{align*}
(A+h)^3 &= (A+h)^2 \cdot (A+h) \\
&= (A^2 + 2Ah + h^2) \cdot (A+h) \\
&= A^3 + A^2h + 2A^2h + 2Ah^2 + Ah^2 + h^3 \\
&= A^3 + 3A^2h + 3Ah^2 + h^3
\end{align*}
Si substituïm $A = x+2$, tenim la fórmula per avançar. 

Anem a la resolució del límit:

\begin{enumerate}
    \item \textbf{Plantegem el límit:}
    \[ f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h} \]
    
    \item \textbf{Substituïm la funció i apliquem l'agrupació $A = x+2$:}
    \[ f'(x) = \lim_{h \to 0} \frac{((x+h)+2)^3 - (x+2)^3}{h} = \lim_{h \to 0} \frac{(A+h)^3 - A^3}{h} \]
    
    \item \textbf{Apliquem el desenvolupament que hem trobat abans:}
    \[ f'(x) = \lim_{h \to 0} \frac{(A^3 + 3A^2h + 3Ah^2 + h^3) - A^3}{h} \]
    
    \item \textbf{Simplifiquem els termes $A^3$:}
    \[ f'(x) = \lim_{h \to 0} \frac{3A^2h + 3Ah^2 + h^3}{h} \]
    
    \item \textbf{Traiem factor comú la $h$ al numerador:}
    \[ f'(x) = \lim_{h \to 0} \frac{h(3A^2 + 3Ah + h^2)}{h} \]
    
    \item \textbf{Cancelem la $h$ i resolem el límit quan $h \to 0$:}
    \[ f'(x) = \lim_{h \to 0} (3A^2 + 3Ah + h^2) = 3A^2 + 3A(0) + (0)^2 = 3A^2 \]
    
    \item \textbf{Desfem el canvi $A = x+2$ per donar el resultat final:}
    \[ f'(x) = \mathbf{3(x+2)^2} \]
\end{enumerate}
\end{enumerate}
