\textbf{1. Expandir logaritmes (D'un a diversos)} \\

Utilitza les propietats del producte, del quocient i de la potència per escriure les expressions següents com a sumes, restes o múltiples de logaritmes:

\begingroup
\setlength{\fboxsep}{10pt}
\noindent\fbox{\begin{minipage}{\dimexpr\textwidth-20pt}
    \textbf{Exemples resolts (Expandir):}
    \begin{itemize}
        \item \textbf{Producte i potència:} $\log_a (x \cdot y^3) = \log_a x + \log_a y^3 = \log_a x + 3\log_a y$
        \item \textbf{Quocient i arrel:} $\log_a \left( \frac{\sqrt{x}}{z} \right) = \log_a \sqrt{x} - \log_a z = \log_a x^{1/2} - \log_a z = \frac{1}{2}\log_a x - \log_a z$
    \end{itemize}
\end{minipage}}
\endgroup
\vspace{0.4cm}
\begin{multicols}{2}
\begin{enumerate}[label=\alph*)]
    \item $\log_2 (4 \cdot x) = $
    \item $\log_3 \left(\frac{x}{9}\right) = $
    \item $\log (x^5) = $
    \item $\log_a (3 \cdot b^2) = $
    \item $\log_5 \sqrt{x} = $ 
    \item $\log \left(\frac{100}{x}\right) = $
    \item \textbf{Repte:} $\log_5 \left(\frac{x^2 \cdot y}{z^3}\right) = $
\end{enumerate}
\end{multicols}

\textbf{2. Condensar logaritmes (De diversos a un de sol)} \\
Ara farem el procés invers. Expressa les operacions següents com un únic logaritme:\\
\begingroup
\setlength{\fboxsep}{10pt}
\noindent\fbox{\begin{minipage}{\dimexpr\textwidth-20pt}
    \textbf{Exemples resolts (Condensar):}
    \begin{itemize}
        \item \textbf{Suma a producte:} $2\log x + \log y = \log x^2 + \log y = \log(x^2 \cdot y)$
        \item \textbf{Resta a quocient:} $\log A - \frac{1}{3}\log B = \log A - \log B^{1/3} = \log A - \log \sqrt[3]{B} = \log \left( \frac{A}{\sqrt[3]{B}} \right)$
    \end{itemize}
\end{minipage}}
\endgroup
\vspace{0.4cm}
\begin{multicols}{2}
\begin{enumerate}[label=\alph*)]
    \item $\log 2 + \log 5 = $
    \item $\log_3 20 - \log_3 4 = $
    \item $4 \log_2 x = $
    \item $\log_a 3 + \log_a b = $
    \item $\frac{1}{2} \log x - \log y = $
    \item $2 \log_5 a + \log_5 b = $
    \item \textbf{Repte:} $2 \log x + 3 \log y - \log z = $
\end{enumerate}
\end{multicols}

\textbf{3. La identitat fonamental} \\
Aplica la propietat $a^{\log_a x} = x$ per simplificar les expressions següents a un sol nombre o variable:\\
\begingroup
\setlength{\fboxsep}{10pt}
\noindent\fbox{\begin{minipage}{\dimexpr\textwidth-20pt}
    \textbf{Exemple resolt (Identitat fonamental):}
    Recorda que si la base de la potència i la base del logaritme són iguals, s'anul·len:
    \begin{itemize}
        \item $10^{\log 7} = 7$ (quan no hi ha base al logaritme, és base 10).
        \item $e^{\ln(x+1)} = x+1$ (el logaritme neperìa $\ln$ té base $e$).
    \end{itemize}
\end{minipage}}
\endgroup
\vspace{0.4cm}
\begin{multicols}{2}
\begin{enumerate}[label=\alph*)]
    \item $3^{\log_3 7} = $
    \item $10^{\log 25} = $
    \item $5^{\log_5 (x+2)} = $
    \item $2^{\log_2 16} = $
    \item $7^{\log_7 (2x)} = $
    \item $4^{\log_4 1} = $
\end{enumerate}
\end{multicols}

\vspace{0.3cm}

\textbf{4. Conceptes clau i anàlisi gràfica} \\
Respon breument a les preguntes següents basant-te en les propietats bàsiques dels logaritmes:
\begin{enumerate}[label=\arabic*.]
    \item Sense fer cap càlcul complex, quant val $\log_{17} 17$? I quant val $\log_{45} 1$?
    \item Imagina que dibuixes la gràfica de $y = \log_2 x$ i la de $y = \log_5 x$. Hi ha un punt exacte de la gràfica per on passaran totes dues funcions. Quin és aquest punt i per què?
\end{enumerate}

\section*{Solucionari}

\textbf{1. Expandir logaritmes}
\begin{multicols}{2}
\begin{enumerate}[label=\alph*)]
    \item $\log_2 4 + \log_2 x = 2 + \log_2 x$
    \item $\log_3 x - \log_3 9 = \log_3 x - 2$
    \item $5 \log x$
    \item $\log_a 3 + \log_a b^2 = \log_a 3 + 2 \log_a b$
    \item $\log_5 x^{\frac{1}{2}} = \frac{1}{2} \log_5 x$ 
    \item $\log 100 - \log x = 2 - \log x$
    \item $\log_5 x^2 + \log_5 y - \log_5 z^3 = 2 \log_5 x + \log_5 y - 3 \log_5 z$
\end{enumerate}
\end{multicols}

\vspace{0.5cm}

\textbf{2. Condensar logaritmes}
\begin{multicols}{2}
\begin{enumerate}[label=\alph*)]
    \item $\log(2 \cdot 5) = \log 10 = 1$
    \item $\log_3 \left(\frac{20}{4}\right) = \log_3 5$
    \item $\log_2 x^4$
    \item $\log_a (3b)$
    \item $\log \sqrt{x} - \log y = \log \left(\frac{\sqrt{x}}{y}\right)$
    \item $\log_5 a^2 + \log_5 b = \log_5(a^2 b)$
    \item $\log x^2 + \log y^3 - \log z = \log \left(\frac{x^2 y^3}{z}\right)$
\end{enumerate}
\end{multicols}

\vspace{0.5cm}

\textbf{3. La identitat fonamental}
\begin{multicols}{2}
\begin{enumerate}[label=\alph*)]
    \item $7$
    \item $25$
    \item $x+2$
    \item $16$
    \item $2x$
    \item $1$
\end{enumerate}
\end{multicols}

\vspace{0.5cm}

\textbf{4. Conceptes clau i anàlisi gràfica}
\begin{enumerate}[label=\arabic*.]
    \item $\log_{17} 17 = 1$. $\log_{45} 1 = 0$. (Per definició: $17^1 = 17$ i $45^0 = 1$).
    \item Ambdues funcions passen pel punt $(1, 0)$. Això passa perquè el logaritme d'1 en qualsevol base sempre és 0 (qualsevol nombre real positiu elevat a zero dona 1).
\end{enumerate}
